Echolocation Helmet

Tasked with teaching a subject on the Virginia SOL to a class of seventh grade students, my group used an Arduino hooked up to an ultrasonic distance sensor and a buzzer to create a helmet that informed the wearer of how close they were to a wall by increasing the rate of the beeps being emitted. The students were able to get a taste of how echolocation works by attempting to navigate an obstacle course while blindfolded and wearing the helmet.

View Project

iBathroom

This project focused around an app idea to eliminate the need to stand in line to use the port-a-johns at tailgates, amusement parks, or concerts by keeping track of a virtual queue, and then notifying you when it was almost their turn to use the bathroom. The user receives a random code, and this code becomes the only working code for the lock on the bathroom door during that person’s allocated time. If the code isn't entered within 30 seconds, your turn is skipped and the next person’s code becomes active. The prototype was built with a large Styrofoam door attached to some PVC pipe to make a bathroom and a Propeller microcontroller controlling the system. Servo motors served as the locks for the door, a screen displayed the next several people in the queue, and a remote was used to enter the code.
